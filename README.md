# Blues Brothers PC game ported for ti89 calculator

Watch the video:

See install instruction into "install" directory

# Controls
ESC : quit the game
 
Shift (black key with an arrow) : grab/launch objects
 
Shift + up : enter doors
 
"+" slows down the game
 
"-" speeds up the game
